# 📁 Proyectos relevantes:

### 📊 Sistema de Indicadores
Proyecto desarrollado para la alcaldía de Córdoba, Quindío. Permite generar indicadores y visualizarlos en gráficos a partir de datos en Excel.

### 📦 Sistema de Gestión de Inventario (Inventrack)
Sistema creado para una fundación que permite gestionar inventarios con formularios dinámicos según el tipo de producto. Incluye alertas para vencimientos y bajo stock.

Proyecto colaborativo que permite la gestión completa de una veterinaria, desde el registro de mascotas hasta el control de medicamentos.
